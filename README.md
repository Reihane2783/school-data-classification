# school-data-classification
A classification project using the School Database dataset from Kaggle. Includes data preprocessing, feature selection, and ML model implementation in Python.
## Objective
To build a machine learning model that classifies different types of data within the school dataset by selecting appropriate features and algorithms.

## Dataset
The dataset provides a comprehensive snapshot of an educational institution's ecosystem, including students, teachers, subjects, and academic performance. It is structured as a relational database designed for efficient data management and information retrieval.

**Source:** https://www.kaggle.com/datasets/bernardnm/great-school?select=Marks.csv
